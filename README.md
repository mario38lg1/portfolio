# Portfolio de Mario López

🚀 **Portafolio personal** — Desenvolvedor Full Stack

Portfolio moderno y responsivo que muestra proyectos, habilidades técnicas y experiencia profesional. Desarrollado con React y diseñado para presentar una experiencia de usuario optimizada.

## ⚙️ Tecnologías

**Frontend:**
- React 18
- React Bootstrap
- React Router v6
- React PDF (para visualización de CV)
- React Icons
- Typewriter Effect (animaciones de texto)

**Hosting:** GitHub Pages / Vercel

**Estilos:** CSS3 personalizado con variables nativas, animaciones y diseño responsivo (Mobile-first)
