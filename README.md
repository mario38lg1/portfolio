🚀 Sitio web oficial en producción: Visitar https://luciamaciasescritora.es

Landing page y portfolio diseñado y desarrollado a medida para presentación literaria y exposición de catálogo de libros. Creado sin frameworks externos (HTML5, CSS3 y JS Vanilla) para maximizar el rendimiento.

⚙️ Tecnologías y Arquitectura
Frontend: HTML5 Semántico, CSS3 (Variables nativas y animaciones), JavaScript puro.
Diseño: Responsive (Mobile-first) con sistema de modales dinámicos.
Infraestructura: Despliegue en GitHub Pages, gestión de Zona DNS, dominio personalizado .es y certificado de seguridad SSL (HTTPS).
